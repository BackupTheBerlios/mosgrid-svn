Certificate Portlet Description

This portlet can be used for handling certificates.
Three certificate formats are supported: SAML Assertion, P12 and PEM files.
SAML assertion is used for submitting jobs in the D-Grid (MoSGrid) infrastructure.
P12 and PEM files are used for accessing files via XTreemFS.

First you can select the option by selecting the appropriate radio button.
If you select the SAML radiobutton, you can do two things:
  1. Upload an existing SAML Assertion.
  2. Generate a SAML Assertion. (The applet appears).
If you select the P12 radiobutton, you can do the following things:
  1. Upload the certificate, which is in pkcs12 format.
  2. Fill in the details needed for the MyProxy server.
If you select the PEM radiobutton, you can do the following things:
  1. Upload the key and public certificate, which is in pem format.
  2. Fill in the details needed for the MyProxy server.
Messages appear at the bottom of the portlet to guide you, such as: "Fill in the details."
After you filled in the required details, you have to press the Upload button.
Then the portal uploads the file(s) for you.
